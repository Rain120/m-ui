## API

| 属性      | 说明                                                         | 类型                      | 默认值     | 版本  |
| :-------- | ------------------------------------------------------------ | ------------------------- | ---------- | ----- |
| type      | 图标类型                                                      | `string`                  | -          | 0.0.1 |
| style     | 设置图标的样式，例如 `fontSize` 和 `color`                   | `CSSProperties`           | -          | 0.0.1 |
| theme     | 图标主题风格。可选实心、描线、双色等主题风格，适用于官方图标 | `filled`\| `outlined`      | `outlined` | 0.0.1 |
| spin      | 是否有旋转动画                                               | `boolean`                 | `false`    | 0.0.1 |
| rotate    | 图标旋转角度                                                 | `number`                  | -          | 0.0.1 |
| component | 控制如何渲染图标，通常是一个渲染根标签为 `<svg>` 的 `React` 组件，**会使 type 属性失效** | `ComponentType<SvgProps>` | -          | 0.0.1 |
